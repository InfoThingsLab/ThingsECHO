AVS_Result AVS_Play_Sound ( AVS_Handle  hHandle,
uint32_t  flags,
void *  pWave,
int32_t  volumePercent 
)

Play a short sound.

It is possible to play only one sound at once, if a sound is already in progress AVS_Play_Sound will return AVS_BUSY In order to limit the foot print and the CPU load, we can produce only up sampling. The sample must always be mono and never have an higher frequency than the native audio device
